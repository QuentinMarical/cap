## ğŸ“ Getting Started

1. Open Garry's Mod and spawn an **Expression 2** chip (found in Wiremod)
2. Right-click the chip and select **"Open Editor"**
3. Copy the code from any `.txt` file in this folder
4. Paste it into the E2 editor
5. Click **"Upload to Chip"**
6. Wire the chip to your CAP entities as described in the script comments
